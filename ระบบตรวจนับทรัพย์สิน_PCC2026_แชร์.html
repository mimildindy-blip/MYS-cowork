ตั้งค่าการแชร์เรียลไทม์ (Firebase)

กรอกค่า Firebase config เพียงครั้งเดียว แล้วทุกคนที่เปิดเว็บนี้จะเห็นผลตรวจนับ รูปภาพ และความคืบหน้าร่วมกันแบบเรียลไทม์ · ค่านี้จะถูกบันทึกไว้ในเครื่องนี้

  1. เข้า console.firebase.google.com → สร้างโปรเจกต์ (ฟรี)
  2. เมนู Build → Realtime Database → Create Database → เลือก Start in test mode
  3. ⚙ Project settings → ส่วน Your apps → กดไอคอน </> (Web) → คัดลอกออบเจกต์ firebaseConfig
  4. วางทั้งก้อนด้านล่าง (ต้องมี apiKey และ databaseURL) แล้วกดบันทึก
